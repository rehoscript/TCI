HTML Files and Structure

About Deita Template

Deita Theme is responsive multi page HTML5 template suitable for agency or business. Deita is retina ready with 16 color styles, awesome LayerSlider, Parallax ready and much more. It is a great way to present your agency at the highest level. It is based on popular Twitter Bootstrap framework. This template uses the latest HTML5 and CSS3 technologies.

Important Notices!
To change color style just open "index.html" and in html head change "css/styles.css" to some of these:

Columns

Template is divided into rows and columns, with respective classes .row and .span*, where rows divide horizontal parts of the page and columns divide each row into smaller vertical sections. Since this template is based on 12-column grid system, you can divide each row into various number of vertical sections as long as they sum up at twelve (12). Look at the examples below for better explanation and some real examples.

Full-width: To achieve a full-width column, use the code below:

1 2 3 4 5

Container

There's one more special class which is being used across all HTML files, which is .container-fluid. This class determines the maximum width of a page and also makes sure that page is center-aligned. It also determines rendering of a template on mobile devices and tablets. This class can have also different backgrounds.

JavaScript Plugins and Scripts

This template comes with these Javascript files:

jquery.js
jQuery is a Javascript library that greatly reduces the amount of code that you must write.

bootstrap.min.js
Twitter bootstrap base JavaScript file, which allows us to use many javascript components like tabs, accordion menus, dropdowns, tooltips and popovers, alert messages and so on. More information and examples can be found at http://twitter.github.com/bootstrap/javascript.html

animate.js
For animate css

jquery.flexslider-min.js
FlexSlider 2 JavaScript file.

jquery.isotope.min.js
A dynamic layout plugin for jQuery. The flip-side of CSS floats.

jquery.fitvids.js
A lightweight, easy-to-use jQuery plugin for fluid width video embeds.

jquery.easing.js
Uses the built in easing capabilities added In jQuery 1.1 to offer multiple easing options

jquery.parallax-1.1.3.js
Parallax

jquery.magnific-popup.min.js
Magnific Popup

js/jquery.sticky.js
Fixed menu on the top of the site.

js/tinynav.min.js
Responsive navigation plugin.

jquery.scrollTo.js
Smooth scrolling of the site.

retina.js
For Retina Images

functions.js
The most important file of Deita template, which defines some basic javascript functions. More information inside JS file.
